# Hot or cold
A game to guess a random number between 1 and an user input.

## Usage
1. Compile
  ```bash
  g++ main.cpp -o hot_cold
  ```

2. Run the game
```bash	
./hot_cold [<guess_limit>]
```

3. Guess a number between 1 and an <guess_limit>
### Parameters
- `-h` or `--help`: Print application usage.
